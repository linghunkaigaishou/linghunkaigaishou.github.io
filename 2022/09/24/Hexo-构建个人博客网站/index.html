Hexo 构建 个人博客系统

发表于 2022-09-24- 更新于 2023-03-10
技术小笔记-Hexo

本地部署Hexo环境

安装NodeJS

安装NodeJS，Hexo是基于NodeJS编写的，所以需要安装NodeJS和npm工具
官网下载地址：https://nodejs.org/zh-cn/download/

下一步，下一步，下一步 。。。。。。

安装Hexo

1、安装Hexo npm install -g hexo-cli
2、查看Hexo版本 hexo -v

发布一个本地网站

1、初始化本地文件夹为网站根目录 hexo init blog （默认为当前文件夹）
2、进入文件夹blog cd blog
3、本地安装 npm install
4、生成静态文件 hexo g
5、启动本地服务器 hexo s （hexo server 的缩写，-p 可指定端口；-i 指定监听的IP地址）
6、浏览器访问 http://localhost:4000

Github上托管hexo

申请Github账号

官网地址：https://github.com/
新手可参考：https://blog.csdn.net/yaorongke/article/details/119086305

创建仓库

1、在 Github 新建一个代码仓库用于保存我们的静态网页
点击Your repositories ,进入仓库页面。

2、修改目录blog下配置文件_config.yml，将deploy参数修改如下：
type: ‘git’
repo: https://github.com/linghunkaigaishou/linghunkaigaishou.github.io.git
branch: master
3、发布网站 hexo g -d

安装Git

Hexo 部署到Github时要用到Git

官网下载页面：https://git-scm.com/downloads

下一步，下一步，下一步 。。。。。。

git config –global user.name “输入你的用户名”
git config –global user.email “输入你的邮箱”
查看user.name指令：git config user.name
查看user.email指令：git config user.email

更换主题

Hexo 主题官网:https://hexo.io/themes/
1、进入网站根目录 cd blog
2、安装主题 git clone https://github.com/rudy-yuan/free2mind.git themes/free2mind
3、修改目录blog下配置文件_config.yml，将theme的参数修改为：theme: free2mind
4、发布网站 hexo g -d

编辑文章

发布文章

1、新建文章 hexo new ‘这是文章标题’
2、清理缓存文件 hexo clean
3、生成静态文件 hexo g （hexo generate的缩写）
4、部署网站 hexo d (hexo deploy的缩写)
5、其中2和3一起操作可以简写为 hexo g -d 或 hexo d -g

文章标签与分类

1、进入站点根目录 cd /blog
2、生成分类页 hexo new page categories
3、对文件index.md，添加type属性type: “categories”
4、生标签页 hexo new page tags
5、对文件index.md，添加type属性type: “tags”
6、文章Front Matter中添加分类和标签属性即可

站点装饰

自定义域名

1、新建CNAME 记录，自定义域名映射github域名
2、github仓库 setting - Pages - Custom domain 将自定义域名保存（用于github.io区分站点）
3、执行hexo g -d 后，github仓库中CNAME配置丢失，在本地source 文件夹下新建一个CNAME文件

安装看板娘小云

板娘模型请参考：https://blog.csdn.net/wang_123_zy/article/details/87181892
项目地址：https://github.com/EYHN/hexo-helper-live2d
1、首先进入Hexo博客根目录安装live2d插件 npm install –save hexo-helper-live2d
2、修改根目录下的 *_config.yml *文件
live2d:
enable: true
scriptFrom: local
pluginRootPath: live2dw/
pluginJsPath: lib/
pluginModelPath: assets/
tagMode: false
debug: false
model:
use: live2d-widget-model-z16
scale: 1
hHeadPos: 0.5
vHeadPos: 0.618
display:
superSample: 2
width: 150
height: 300
position: right
hOffset: 0
vOffset: -20
mobile:
show: true
scale: 0.5
react:
opacityDefault: 0.7
opacityOnHover: 0.2
3、进入博客根目录输入以下命令安装：npm install live2d-widget-model-z16
